matplolib
Another great thing about pandas is that it integrates with Matplotlib, so you get the ability to plot directly off DataFrames and Series.
If you don't have matplotlib installed then use: pip install matplotlib.

In[36]:
# set font and plot size to be larger
plt.rcParams.update({'font.size': 20, 'figure.figsize': (10, 8)})

Which type of plot to use
For categorical variables utilize Bar Charts* and Boxplots.

For continuous variables utilize Histograms, Scatterplots, Line graphs, and Boxplots.

Create a scatter plot
What's with the semicolon?
It's not a syntax error, just a way to hide the <matplotlib.axes._subplots.AxesSubplot at 0x26613b5cc18> output when plotting in Jupyter notebooks.

In[37]:
movies_df.plot(kind='scatter', x='rating', y='revenue_millions', title='Revenue (millions) vs Rating');

Create a simple Histogram
This Histogram is based on a single column, we can call plot on a column:

In[38]:
movies_df['rating'].plot(kind='hist', title='Rating');

Boxplot
A Boxplot is a graphical representation of the interquartile range.

In[40]:
movies_df['rating'].plot(kind="box");

Recreate the rating_category columns

In[41]:
movies_df["rating_category"] = movies_df["rating"].apply(lambda x: 'good' if x >= 8.0 else 'bad')

Boxplot
By combining categorical and continuous data, we can create a Boxplot of revenue that is grouped by the Rating Category we created above:

In[42]:
movies_df.boxplot(column='revenue_millions', by='rating_category');
